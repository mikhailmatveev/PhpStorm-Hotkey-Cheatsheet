# PhpStorm | Список горячих клавиш для повышения скорости работы

## Основные

|Сочетание клавиш|Оригинальное название команды|Назначение|
| --- | --- | --- |
|`Shift` `Shift`|Search everywhere|Быстрый поиск по всем категориям|
|`Ctrl`+`Shift`+`A`|Find Action|Быстрый поиск действия (команды меню)|
|`Alt`+`0`...`9`|Open corresponding tool window|Открыть соответствующую панель инструментов|
|`Ctrl`+`Alt`+`F11`|Toggle full screen mode|Включить/отключить полноэкранный режим|
|`Ctrl`+`Shift`+`F12`|Toggle maximizing editor|Развернуть/свернуть окно редактора. Позволяет свернуть боковые панели инструментов и максимально увеличить видимую часть текстового редактора|
|`Alt`+`Shift`+`F`|Add to Favorites|Добавить в список избранного|
|`Alt`+`Shift`+`I`|Inspect current file with current profile|Открыть инспектор ошибок и предупреждений для текущего файла и текущего профиля|
|`Ctrl`+`Alt`+`S`|Open Settings dialog|Открыть настройки|
|`Ctrl`+`Tab`|Switch between tabs and tool window|Переключение между вкладками и панелями инструментов|

## Редактирование

### Стандартные команды

|Сочетание клавиш|Оригинальное название команды|Назначение|
| --- | --- | --- |
|`Ctrl`+`Space`|Basic code completion|Автодополнение кода|
|`Alt`+`Enter`|Show intention actions and quick-fixes|Универсальный контекстный помощник. Отображает контекстное меню с доступными действиями, в зависимости от того, где находится курсор. Включает в себя: <ul><li>Импорт класса под курсором в секцию use;</li><li>Обновление блока комментариев по сигнатуре метода</li><li>Добавление методов интерфейса, которые необходимо реализовать</li><li>Замена кавычек для строк.</li><ul>
|`Ctrl`+`P`|Parameter info (within method call)|Инфо о передаваемом параметре в функцию/метод из места вызова|
|`Ctrl`+`Q`|Quick documentation lookup|Быстрый просмотр документации по конкретной сущности в позиции курсора|
|`Ctrl`+mouse over code|Brief Info|Краткая информация по конкретной сущности в позиции курсора|
|`Alt`+`Insert`|Generate code...|Сгенерировать фрагмент кода|
|`Ctrl`+`O`|Override methods|Переопределить методы в текущем классе|
|`Ctrl`+`I`|Implement methods|Реализовать методы в текущем классе|
|`Ctrl`+`Alt`+`T`|Surround with... (if..else, try..catch, for, etc.)|Окружить блок кода конструкцией (if..else, try..catch, for, и т.д.)|
|`Ctrl`+`/`|Comment/uncomment with line comment|Закомментировать/Раскомментировать строку/блок кода однострочным комментарием|
|`Ctrl`+`Shift`+`/`|Comment/uncomment with block comment|Закомментировать/Раскомментировать строку/блок кода многострочным комментарием|
|`Ctrl`+`W`|Select successively increasing code blocks|Последовательное выделение строки, блока в позиции курсора|
|`Ctrl`+`Shift`+`W`|Decrease current selection to previous state|Последовательная отмена выделения строки, блока в позиции курсора|
|`Ctrl`+`Alt`+`L`|Reformat code|Автоформат кода во всем файле, либо только выделенного фрагмента|
|`Ctrl`+`Alt`+`I`|Auto-indent line(s)|Автоматическое выравнивание отступов строк, либо выделенного фрагмента кода|
|`Ctrl`+`D`|Duplicate current line or selected block|Дублировать строку либо выделенный фрагмент в позиции курсора|
|`Ctrl`+`Y`|Delete line at caret|Удалить строку либо выделенный фрагмент в позиции курсора (так же действие "Отменить")|
|`Ctrl`+`Shift`+`J`|Smart line join (HTML and JavaScript only)|Поочередное удаление переносов строк ниже текущей позиции курсора|
|`Ctrl`+`Enter`|Start new line|Добавить новую строку выше позиции курсора и переместить курсор в позицию новой строки|
|`Ctrl`+`Shift`+`Enter`|Start new line|Добавить новую строку ниже позиции курсора и переместить курсор в позицию новой строки|
|`Ctrl`+`Shift`+`U`|Toggle case for word at caret or selected block|Изменить регистр символов в слове или выделенном фрагменте текста|
|`Ctrl`+`Shift`+`]`/`[`|Select till code block end/start|Выделить фрагмент текста от текущей позиции курсора до конца/начала блока кода|
|`Ctrl`+`Delete`|Delete to word end|Удалить часть слова от позиции курсора к концу слова|
|`Ctrl`+`Backspace`|Delete to word start|Удалить часть слова от позиции курсора к началу слова|
|`Ctrl`+`+`/`-`|Expand/collapse code block|Развернуть/свернуть блок кода|
|`Ctrl`+`F4`|Close active editor tab|Закрыть текущую вкладку редактора|
|`Ctrl`+`Shift`+`V`|Paste from history|Специальная вставка|

### Дополнительные команды

|`Shift`+`Tab`|Уменьшить отступ|

|Сочетание клавиш|Оригинальное название команды|Назначение|

|`Ctrl`+`G`|Go to line|Перейти к строке с номером|
|`Alt`+`Right`/`Left`|Go to next/previous editor tab|Переключиться на следующую/предыдущую вкладку|
|`Esc`|Go to editor (from tool window)|Вернуться в окно редактора|
|`Ctrl`+`E`|Recent files popup|Открыть окно со списком последних измененных файлов|
|`Ctrl`+`Alt`+`Left`/`Right`|Navigate back/forward||
|`Ctrl`+`Shift`+`Backspace`|Navigate to last edit location|Вернуться на предыдущее место правки|
|`Alt`+`F1`|Select current file or symbol in any view||
|`Ctrl`+`B`/`Ctrl`+`Click`|Go to declaration|Перейти в объявление сущности|
|`Ctrl`+`U`|Go to super-method/super-class|Перейти к методу-предку/классу-предку|
|`Alt`+`Up`/`Down`|Go to previous/next method|Перейти к предыдущему/следующему методу|
|`Ctrl`+`]`/`[`|Move to code block end/start|Перейти в конец/начало блока|
|`F2`/`Shift`+`F2`|Next/previous highlighted error|Перейти к следующей/предыдущей подсвеченной в коде ошибке|
|`F4`/`Ctrl`+`Enter`|Edit source/View source|Редактировать код/Просмотр кода|
